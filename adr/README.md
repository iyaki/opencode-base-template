# Architecture Decision Records

This directory stores architecture decisions that affect how this repository is built and maintained.

Conventions:

- Filename format: `YYYY-MM-DD-title-with-dashes.md`
- Status values: `proposed`, `accepted`, `rejected`, `deprecated`, `superseded`
- New architecture defaults and exceptions should be documented here.
